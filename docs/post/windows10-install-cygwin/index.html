What Is Cygwin? Get that Linux feeling - on Windows

这个是Cygwin项目主页上的简介，安装它，可以得到一个类似Linux的Shell环境，在其中你可以使用绝大部分Linux软件和功能。如Gcc,Make,Vim,Emacs等等，作为一个php开发者，使用Cygwin的目的是为了运行Swoole，接下来，我将简单分享下安装Cygwin的流程和我们安装swoole需要的组件包。

一、从项目主页下载Cygwin安装包

这个安装包不单单只是安装Cygwin，还能用来安装我们需要的组件包。

二、安装Cygwin

  1. 选择组件包的下载方式 这里因为我使用了网络工具，网络环境相对来说比较好，所以我选择Install from Internet。

  2. 选择Cygwin的安装路径 安装包提示我们最好选择在盘符的根目录下作为Cygwin的运行目录。

  3. 选择组件包的下载路径

  4. 选择链接方式 这里就根据你自己的网络环境来选择对应的链接方式。

  5. 选择下载组件包的镜像地址 选择下载组件包的镜像地址； 这里也是根据网络环境，国内最好选择网易的镜像源，下载速度快且稳定。

  6. 选择安装组件包 这里就根据你开发环境的需要来安装组件包。

  7. 执行安装组件包和Cygwin 点击下一页后执行安装Cygwin和组件包，完成后就可以打开Cygwin64 Terminal来体验了。

三、安装组件包

这里我列举出我们运行php运行环境和编译Swoole所需要的组件包列表。 这一步，可以在执行安装组件包和Cygwin上进行；特别要提示的是，我们一旦选择好php的版本后，php相关组件包的安装都需要对应好版本。

以上步骤完成后，就可以在Cygwin上进行php开发了，enjoy🤗！！
