> 在业务开发中，对于用户上传的文件需要进行安全校验，*Yii*本身提供了*FileValidator*的文件校验类；但是文件校验类对于word、excel、powerpoint的校验只是针对Microsoft，而wps的文件格式并不支持，因此需要重写文件校验类来达到我们的需求。

通过断点调试，得知*FileValidator*是通过获取上传文件的*MimeType*和后缀，然后将后置通过*yiisoft/yii2/helpers/mimeTypes.php*中返回的数组获取到对应*Yii*设定的*MimeType*，对比是否一致，如果不一致则抛出校验错误。

## 一、重写上传校验类
因此重写的思路就是在*Yii*进行系统判断前，我们新增WPS特殊的格式的校验，如果命中就直接返回验证通过，代码如下：
```php
<?php

namespace app\validators;

use yii\validators\FileValidator as Base;
use yii\helpers\FileHelper;

class FileValidator extends Base
{

    protected $extra = [
        "docx" => "application/zip",
        "et" => "application/vnd.ms-excel",
        "xlsx" => "application/zip",
        "ett" => "application/vnd.ms-excel"
    ];

    protected function validateExtension($file)
    {
        $extension = mb_strtolower($file->extension, 'UTF-8');

        if ($this->checkExtensionByMimeType) {

            $mimeType = FileHelper::getMimeType($file->tempName, null, false);
            if ($mimeType === null) {
                return false;
            }

            $extensionsByMimeType = FileHelper::getExtensionsByMimeType($mimeType);

            if($this->extra[$extension] == $mimeType){
                return true;
            }

            if (!in_array($extension, $extensionsByMimeType, true)) {
                return false;
            }
        }

        if (!in_array($extension, $this->extensions, true)) {
            return false;
        }

        return true;
    }

}
```
上面的代码块中，``$extra``是我们对于WPS的文件设定的额外校验规则，在方法``validateExtension``中，判断后缀名和*MimeType*是否一致是的话就放行。

## 二、表单模型中调用
在调用时要引用我们重写的工具类，在表单校验规则中指定为``FileValidator::className()``，而不是原先的``file``；
```php
<?php

use yii\base\Model;
use yii\helper\ArrayHelper;
use app\validators\FileValidator;

class Example extends Model{

    public function rules()
    {
    
        $parent_rules = parent::rules(); // TODO: Change the autogenerated stub
    
        $rules = [
            [
                'files',
                FileValidator::className(),
                'skipOnEmpty' => false,
                'extensions' => 'png,jpg,jpeg,gif,doc,dot,docx,xls,xlt,xlsx,xlsm,ppt,pot,pps,pptx,pdf'
            ],
        ];
    
        return ArrayHelper::merge(
            $parent_rules,
            $rules
        );
    
    }
    
}
```
